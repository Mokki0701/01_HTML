style 태그 내용 적용 확인

block 형식

        - 화면(공간)을 수직 분할
        (세로 화면을 가로로 잘라서 층(행)을 나눔) -> 한층을 행이라고 함

        - 한 행(한 층)을 모두 차지함

        -width, height 와 같은 크기 지정 속성 사용 가능

        -block 형식의 태그

        h1~h6(제목), p/pre(문단), hr(수평선), div(영역분할), 시맨틱(div 태그에 이름 넣은 태그)

h3는 block 형식 태그 입니다

        block 형식은 작성된 내용이 짧더라도 
        한 줄을 모두 차지하려는 성질때문에 다음 태그 내용은 다음 줄에 작성됩니다
첫 번째 div
두 번째 div
세 번째 div

inline 형식

        - 화면을 수평 분할
        (가로 화면을 세로로 잘라서 열(호)를 나눔)

        -한 층(행)을 모두 차지하지 않음
         -> 내부에 작성된 내용 크기 많큼만 차지하려는

        - width, height 와 같은 크기 지정 속성 사용 불가 ! 
        
        -inline 형식의 태그
        b(굵게), i(기울기), mark(글배경 칠하기), span(영역 분할) 등

inline 형식은 행을 분할하는 성질을 가지고 있어 줄이 바뀌지 않고 계속 옆으로 작성된다
첫 번째 span 두 번째 span 세 번째 span

영역 지정 방식의 차이점

  • block 형식 : 사각형 박스 형태로 영역을 지정
  • inline 형식 : 내부에 작성된 내용(content) 단위로 영역을 지정함

inline-block 형식

        - inline의 특징 중 수평분할(줄을 세로로 자름, 열 나눔) 
        - block의 특징 중 크기 지정 가능 

        - 대표적인 태그 : img태그, iframe 태그

        * ifram 태그
        - 웹 문서(html) 내부에 또 다른 웹 문서를 추가하는 태그

radio, checkbox

    - 여러 연관된 값을 묶어서 선택하는 경우에 사용

    - radio : 여러 값 중 하나 선택
      ex) 성별 

    - checkbox : 여러 값 중 원하는 모든 것 선택
      ex) 상세검색, 관심분야, 취미

    
        연관된 radio, checkbox는 같은 name 속성 값을 가져야 한다!
